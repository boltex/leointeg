LeoInteg

Leo Editor Integration 0

Use Leo to program with directed acyclic graphs, along with section-references, clones, file derivation and the Mulder/Ream update algorithm.

LeoInteg is an open-source extension created by Félix that integrates Leo into Visual Studio Code.

Welcome to LeoInteg 0.1.8
